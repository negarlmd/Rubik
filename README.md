# 3D Rotating Cube

A simple and interactive 3D cube built using pure HTML and CSS3 transforms.

## 📌 Project Overview

This project demonstrates the power of CSS3 3D transformations by creating a rotating cube with six faces. Each face represents a different web development technology.

The cube rotates smoothly when hovered, providing an engaging visual effect without using JavaScript.

## ✨ Features

- Pure HTML & CSS
- 3D Cube Design
- CSS3 Transformations
- Hover Rotation Animation
- Preserve-3D Effect
- Responsive Center Alignment
- No JavaScript Required

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Transforms
- CSS Transitions
- 3D Animation

## 🚀 How It Works

The project uses:

- `transform-style: preserve-3d`
- `translateZ()`
- `rotateX()`
- `rotateY()`
- `rotate3d()`
- `transition`

to create and animate a realistic 3D cube effect.
